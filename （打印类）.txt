题目描述
液晶屏上，每个阿拉伯数字都是可以显示成 3×5 的点阵的（其中 X 表示亮点，. 表示暗点）。现在给出数字位数（不超过 100）和一串数字，要求输出这些数字在显示屏上的效果。数字的显示方式如同样例输出，注意每个数字之间都有一列间隔。

输入格式
无

输出格式
无

输入输出样例
输入 #1复制
10
0123456789
输出 #1复制
XXX...X.XXX.XXX.X.X.XXX.XXX.XXX.XXX.XXX
X.X...X...X...X.X.X.X...X.....X.X.X.X.X
X.X...X.XXX.XXX.XXX.XXX.XXX...X.XXX.XXX
X.X...X.X.....X...X...X.X.X...X.X.X...X
XXX...X.XXX.XXX...X.XXX.XXX...X.XXX.XXX





#include<stdio.h>
char a[10][5][4] = {
	"XXX","X.X","X.X","X.X","XXX",//0
	"..X","..X","..X","..X","..X",//1
	"XXX","..X","XXX","X..","XXX",//2
	"XXX","..X","XXX","..X","XXX",//3
	"X.X","X.X","XXX","..X","..X",//4
	"XXX","X..","XXX","..X","XXX",//5
	"XXX","X..","XXX","X.X","XXX",//6
	"XXX","..X","..X","..X","..X",//7
	"XXX","X.X","XXX","X.X","XXX",//8
	"XXX","X.X","XXX","..X","XXX",//9
};
int b[100];
int main()
{
	int i, j, n;
	scanf("%d", &n);
	for (i = 0; i < n; i++) 
		scanf("%1d", &b[i]);
	for (j = 0; j < 5; j++) {
		for (i = 0; i < n; i++) {
			printf("%s", a[b[i]][j]);
			if (i != n - 1) printf(".");
		}
		puts(" ");
	}
	return 0;
}
