给出一个不大于 9 的正整数 nn，输出 n\times nn×n 的蛇形方阵。

从左上角填上 1 开始，顺时针方向依次填入数字，如同样例所示。注意每个数字有都会占用 3 个字符，前面使用空格补齐。

输入格式
无

输出格式
无

输入输出样例
输入 
4
输出 
  1  2  3  4
 12 13 14  5
 11 16 15  6
 10  9  8  7



#include<stdio.h>
int a[10][10];
int main()
{
	int n, i=1, j=0,k=1;
	scanf("%d",&n);
	while (k <= n * n) {
		while (j < n&&!a[i][j+1])  a[i][++j] = k++; 
		while (i < n&&!a[i+1][j])  a[++i][j] = k++; 
		while (j > 1&&!a[i][j-1])  a[i][--j] = k++; 
		while (i > 1&&!a[i-1][j])  a[--i][j] = k++;
	}
	for (i = 1; i <= n; i++) {
		for (j = 1; j <= n; j++)
			printf("%3d", a[i][j]);
		puts(" ");
	}
}
